# ⛽ MuDiesel

Encuentra las gasolineras más baratas cerca de ti en España.

## Funcionalidades

- Geolocalización automática
- Precios en tiempo real (API del Ministerio de Industria y Turismo)
- Filtro por combustible: Diésel, Gasolina 95, Gasolina 98
- Radio de búsqueda: 5 / 10 / 25 km
- Ordenación por precio o cercanía
- Navegación directa a Google Maps

## Demo

👉 [sora-com.github.io/MuDiesel](https://sora-com.github.io/MuDiesel)

## Stack

HTML + CSS + JavaScript vanilla. Sin dependencias. Un solo archivo.

## Datos

Precios obtenidos de la [API REST del Ministerio de Industria y Turismo](https://sedeaplicaciones.minetur.gob.es/ServiciosRESTCarburantes/PreciosCarburantes/). Se actualizan automáticamente cada 5 minutos.
